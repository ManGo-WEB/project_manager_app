# Бэклог фич: Projects Manager

Список планируемых фич для реализации. Статусы: ожидает / одобрено / в работе / реализовано / отклонено.

---

## 1. Хронология активности проекта (Activity Timeline)
**Статус:** ожидает

Лента последних действий на странице проекта: коммиты, изменения статусов, добавление этапов. Источник данных — git log + отслеживание изменений markdown-файлов. Позволяет быстро понять, что происходило с проектом за последнее время.

---

## 2. Заметки к проекту (Project Notes)
**Статус:** ожидает

Возможность добавлять быстрые заметки к проекту через UI — идеи, решения, ссылки. Хранятся в `docs/notes.md`. Не требуют структуры, в отличие от PRD или плана. Полезны для фиксации мыслей на лету.

---

## 3. Дедлайны и сроки
**Статус:** ожидает

Добавление дат дедлайнов к этапам в PLAN.md. На дашборде — индикация просроченных и приближающихся дедлайнов (цветовая маркировка). На странице проекта — таймлайн с датами.

---

## 4. Экспорт отчёта по проекту (PDF/Markdown)
**Статус:** ожидает

Генерация сводного отчёта по проекту: описание, статус, план с прогрессом, текущие этапы. Экспорт в PDF или единый markdown-файл. Полезно для презентации статуса проекту или сохранения снапшота.

---

## 5. Шаблоны проектов
**Статус:** ожидает

Возможность создавать и сохранять шаблоны проектов с предзаполненным планом и этапами. Например: "Веб-приложение", "Landing page", "Бот" — каждый с типовым набором этапов. При создании проекта — выбор шаблона из списка.

---

## 6. Статистика и аналитика
**Статус:** ожидает

Страница с общей статистикой: количество проектов по статусам (pie chart), средний прогресс, количество завершённых за месяц, активность по дням. Помогает видеть общую картину продуктивности.

---

## 7. Быстрые действия на дашборде (Quick Actions)
**Статус:** ожидает

Контекстное меню на карточке проекта (правый клик или кнопка "..."):  архивировать, сменить статус, открыть в проводнике, открыть в VS Code, скопировать путь. Ускоряет рутинные операции без захода внутрь карточки.

---

## 8. Уведомления и напоминания
**Статус:** ожидает

Система напоминаний: проект не обновлялся N дней, приближается дедлайн, есть незакоммиченные изменения. Отображаются как бейджи на дашборде или тост-уведомления. Помогает не забывать о проектах.

---

## 9. Связи между проектами (Dependencies)
**Статус:** ожидает

Возможность указать, что один проект зависит от другого или связан с ним. Отображается на странице проекта как список связанных проектов с кликабельными ссылками. Полезно когда несколько проектов работают в связке.

---

## 10. Мультиязычный интерфейс (i18n)
**Статус:** ожидает

Переключение языка интерфейса: русский / английский. Данные проектов остаются на языке, на котором написаны — переводится только UI (кнопки, заголовки, подписи). Полезно при работе в международной среде или для портфолио.
