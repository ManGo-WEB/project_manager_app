# План реализации: Projects Manager

## Обзор

План разбит на 6 крупных этапов. Каждый этап — логически завершённый блок, после которого можно оценить результат. Этапы выполняются последовательно.

---

- [x] **Этап 1: Проектирование** — Составление PRD, проведение интервью, определение архитектуры и стека технологий
- [x] **Этап 2: Инициализация проекта** — Настройка Next.js, TypeScript, Tailwind CSS, структура папок приложения, базовая конфигурация
- [x] **Этап 3: Backend — API и работа с данными** — API Routes для чтения/записи markdown-файлов, парсинг frontmatter, бизнес-логика (статусы, процент выполнения)
- [x] **Этап 4: Frontend — интерфейс** — Дашборд с карточками проектов, страница проекта, модальные окна, поиск, фильтрация, сортировка, создание проекта
- [x] **Этап 5: Тема и визуальная доработка** — Тёмная/светлая тема, финальная стилизация, адаптивность, polish UI
- [x] **Этап 6: Дополнительный функционал** — Базовое редактирование через UI, архив проектов, инициализация Git при создании проекта
